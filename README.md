# TNC Mobil – Mini Katalog Uygulaması

Kullanıcıların sahte ürün verisi (FakeStore API) üzerinden ürünleri listeleyip detaylarını görüntüleyebildiği, sepete ekleme özelliği olan basit bir mini katalog uygulaması.

## Flutter Sürümü

- Flutter **3.41.4** (stable)

## Çalıştırma Adımları

1. Depoyu/klasörü makinenize indirin veya açın:
   ```bash
   cd "tnc mobil"
   ```
2. Gerekli paketleri indirin:
   ```bash
   flutter pub get
   ```
3. Bir cihaz veya emülatör çalıştırın (Android Emulator, iOS Simulator veya fiziksel cihaz).
4. Uygulamayı başlatın:
   ```bash
   flutter run
   ```
5. Gerekirse belirli bir cihaza çalıştırmak için:
   ```bash
   flutter devices        # cihaz id'lerini gör
   flutter run -d <cihaz_id>
   ```

Bu adımlar tamamlandıktan sonra ana ekranda ürün listesini, detay sayfasında ürün açıklamasını ve sepette seçili ürünleri görüntüleyebilirsiniz.
